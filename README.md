# Memory Map Primitives

![Development Status](https://img.shields.io/badge/status-active--development-blue.svg)

Memory-mapping (mmap) vocabulary — `Memory.Map` namespace with `Region`, `Protection`, `Sharing`, `Advice`, `Anonymous`, `File`, `Options`, `Access`, `Safety`, `Error` types.

Sibling extraction of [swift-memory-primitives](https://github.com/swift-primitives/swift-memory-primitives). Syscall implementations live in platform packages — POSIX in [swift-iso-9945](https://github.com/swift-standards/swift-iso-9945) (`extension Memory.Map`), Windows in [swift-windows-standard](https://github.com/swift-standards/swift-windows-standard).
